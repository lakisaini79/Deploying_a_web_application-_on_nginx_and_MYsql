# Web Application Deployment with Nginx and MySQL

This project demonstrates deploying a web application using DevOps best practices.
The focus of this repository is infrastructure setup, containerization, and deployment
automation rather than application development.

## Architecture Overview
- Nginx is used as a reverse proxy and web server
- Application runs inside a Docker container
- MySQL is used as the relational database
- Docker Compose manages multi-container setup

## Tech Stack
- Web Server: Nginx
- Database: MySQL
- Containerization: Docker
- Orchestration: Docker Compose
- Operating System: Linux (Ubuntu)
- Cloud Platform: AWS EC2 (optional)

## Features
- Reverse proxy configuration using Nginx
- Persistent MySQL database using Docker volumes
- Environment-based configuration
- Production-ready container setup
- Scalable and modular architecture
